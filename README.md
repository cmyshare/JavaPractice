#  目录结构
```md
JavaPractice
├── .idea                   # IntelliJ IDEA 项目配置文件
├── ActivitiDemo            # 流程引擎 Activiti 实践
├── ApachePoiDemo           # Excel 文档处理（Apache POI）
├── CodeGenerateDemo        # 代码生成器实践
├── CodeGenerateRyDemo      # 基于 RuoYi 框架的代码生成器
├── EasyExcelDemo           # 高性能 Excel 处理（EasyExcel）
├── ImageDemo               # 图片处理（如缩略图、水印等）
├── LeetCodeDemo            # LeetCode 算法题练习
├── MongodbDemo             # MongoDB 数据库操作
├── MysqlDemo               # MySQL 数据库操作
├── NettyDemo               # Netty 网络编程实践
├── OfficeDemo              # Word/PPT/Excel 综合处理
├── RabbitMQDemo            # RabbitMQ 消息队列实践
├── RedisDemo               # Redis 缓存与数据结构使用
├── RocketMQDemo            # RocketMQ 消息队列实践
├── RuleEngineDemo          # 规则引擎（如 Drools）实践
├── ScheduledTaskDemo       # 定时任务（Quartz 或 Spring Task）
├── ToolDemo                # 工具类封装（如工具函数、通用组件）
├── WebSocketDemo           # WebSocket 实时通信
├── XxlJobDemo              # 分布式定时任务调度（XXL-JOB）
├── .gitignore              # Git 忽略文件配置
└── pom.xml                 # Maven 项目根配置文件（多模块聚合）
```
